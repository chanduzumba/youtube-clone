# 🎬 YouTube Clone (MERN Stack)

A full-stack YouTube Clone built using the MERN stack that replicates the core functionality of YouTube, including authentication, video uploads, channels, comments, likes/dislikes, subscriptions, search, and responsive UI.

---

## 🚀 Features

- User Authentication
- JWT Authorization
- Create Channel
- Upload Videos
- Edit/Delete Videos
- Watch Videos
- Search Videos
- Like/Dislike Videos
- Comment System
- Like/Dislike Comments
- Channel Pages
- Responsive Design

---

## 🛠 Tech Stack

## Installation

### Clone repository

```bash
git clone https://github.com/chanduzumba/youtube-clone.git
```

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
