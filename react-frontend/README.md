# YouTube Clone Frontend

React frontend for the YouTube Clone.

## Tech Stack

- React
- Vite
- Redux Toolkit
- React Router
- Axios
- Tailwind CSS
- React Icons

---

## Features

- Responsive UI
- Authentication
- Home Feed
- Watch Page
- Search
- Channel Page
- Profile
- Upload Video
- Comments
- Likes & Dislikes

---

## Installation

```bash
npm install
```

Start

```bash
npm run dev
```

---
